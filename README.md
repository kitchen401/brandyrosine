# Brandy Rosine Memorial

The website for **brandyrosine.com** — a memorial honoring Brandy Stevens-Rosine
(November 30, 1992 – May 17, 2012).

This is a simple, static website: plain HTML, CSS, and images. There is no
software to build or maintain, no database, and no monthly platform fee. It is
designed to be hosted for free on **GitHub Pages**.

## What's here

| Page | File |
|------|------|
| Home | `index.html` |
| Her Story | `her-story.html` |
| Her Words (Brandy's writing) | `her-words.html` |
| The Tragedy | `the-tragedy.html` |
| Remembering Her | `remembering.html` |
| Save the Date (2027) | `save-the-date.html` |
| Contact | `contact.html` |

- `assets/style.css` — all of the site's styling (colors, fonts, layout).
- `images/` — all photos, at full resolution.
- `source-content/` — a plain-text archive of every page and blog post salvaged
  from the original Wix site. Kept as a backup; not part of the live site.

## How to make a change

1. Open the relevant `.html` file in any text editor.
2. Edit the text between the tags.
3. Save, commit, and push to GitHub — the live site updates automatically.

The **Save the Date** wording lives in `save-the-date.html` (and a banner on
`index.html` and `remembering.html`). When the 2027 date is confirmed, update
those three spots and remove the "date to be confirmed" note.

## Hosting

Hosted on GitHub Pages. The custom domain (brandyrosine.com) is pointed here via
DNS at the domain's registrar (Bluehost). See the project notes for the
step-by-step domain switch.
